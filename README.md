## Available Tools

### 1. Lighthouse Auto Scanner (`lighthouse_auto.py`)
Automated security auditing tool that performs:
- Google Lighthouse performance audits
- API key and credential extraction (AWS, Google, GitHub, Stripe, etc.)
- Private key detection (RSA, SSH, PGP)
- Web crawling and sensitive data analysis

**Usage:**
```bash
# Basic scan
python lighthouse_auto.py https://example.com

# Advanced scan with custom settings
python lighthouse_auto.py https://example.com --depth 3 --output ./reports
```

See [LIGHTHOUSE_README.md](LIGHTHOUSE_README.md) for detailed documentation.

### 2. Packet Analyzer (`matrixs1.0.py`)
Simple Wireshark-like tool for packet capture and analysis.

### 3. Nmap GUI Scanner (`TESTNMAP.py`)
Graphical interface for running Nmap security scans.

### 4. StakeUS GLD Converter (`stakeus_gld_converter.py`)
Cryptocurrency staking calculator and cash converter.
